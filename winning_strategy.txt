WINNING STRATEGY - SLOT BUILDER

Goal:
- Reach stable passive income quickly
- Complete milestones/quests efficiently
- Hit victory target with balanced structure growth

==================================================
1) OPENING (FIRST 3-5 MINUTES)
==================================================
Priority:
1. Keep enough Energy to continue spinning
2. Build a balanced base early

Opening sequence:
- Spin until you can build Sawmill + Quarry
- Build Reactor as soon as possible so Energy economy stabilizes
- Build Forge after Reactor to unlock reliable Metal flow

Checkpoint before heavy upgrades:
- At least 1 of each structure
- Energy no longer crashing to zero

Rules:
- Do not spam spins if Energy is dropping faster than Reactor output
- Avoid overbuilding one structure too early because scaling costs climb

==================================================
2) EARLY ECONOMY STABILIZATION
==================================================
Structure ratio target:
- Keep near: Sawmill : Quarry : Forge : Reactor = 2 : 2 : 1 : 2

Why:
- Wood/Stone feed most expansion costs
- Reactor protects spin uptime
- Forge supports upgrades and rerolls

Spin discipline:
- Spin in bursts, then pause to let passive tick
- If Energy dips, stop spinning and build Reactor or wait for passive recovery

==================================================
3) UPGRADE ORDER (MOST CONSISTENT)
==================================================
Recommended priority:
1. Efficient Reels (lower spin cost)
2. Automation Grid (increase passive output)
3. Symbol Compression (better spin base gains)
4. Lucky Core (stronger pair/jackpot payouts)

Upgrade pattern:
- Buy 1 level in each top priority path first
- Then alternate Efficient Reels + Automation Grid
- Add Symbol Compression when spins feel under-rewarding
- Add Lucky Core after spin economy is stable

==================================================
4) QUEST OPTIMIZATION
==================================================
Quest board rules:
- Prefer quests aligned with what you are already doing
- Reroll only when all offered quests are off-plan

Reroll economics:
- Cost is 20 Energy + 20 Metal
- If rerolling slows key build/upgrade timing, skip it
- Best reroll window: after Forge production ramps up

Fast completion trick:
- If all quests are done, board refresh accelerates
- Use this to chain reward bursts during mid-game

==================================================
5) BIOME STRATEGY
==================================================
General approach:
- Meadow for safe consistency
- Volcanic when passive-focused and you can handle event volatility
- Arctic when you need stronger spin returns
- Neon when fully online and chasing fast endgame scaling

Practical rule:
- Switch biomes only when you understand your current bottleneck
- If uncertain, stay in Meadow until your economy is resilient

==================================================
6) EVENT MANAGEMENT
==================================================
Positive events:
- Convert gains into structures/upgrades immediately

Negative events:
- If Energy penalty events chain, reduce spin frequency
- Recover with Reactor build and passive wait cycles

Best defense:
- Balanced structure base + enough Reactor count

==================================================
7) MID-GAME PUSH
==================================================
Targets:
- 3-4 of each structure
- 6+ total upgrade levels
- Momentum where passive/sec keeps rising each minute

Execution:
- Spend in this order when uncertain:
  1) Reactor if Energy unstable
  2) Cheapest high-impact structure for bottleneck resource
  3) Efficient Reels/Automation Grid
  4) Selective spins for quest progress and bonus spikes

==================================================
8) ENDGAME CONVERSION TO VICTORY
==================================================
Victory path:
- Push to 5 of each structure
- Maintain enough total resources to hit threshold

Closing checklist:
- Do not over-spin in endgame if passive already carries growth
- Build missing structure counts first
- Use quests/milestones as finish accelerators

==================================================
9) COMMON MISTAKES TO AVOID
==================================================
- Over-spinning while Energy collapses
- Ignoring Reactor and getting spin-locked
- Spending reroll cost too early
- Hyper-focusing one structure while costs spike
- Chasing jackpots instead of passive scaling

==================================================
10) SIMPLE WIN FORMULA
==================================================
If unsure, follow this formula:
- 1 of each structure quickly
- Prioritize Efficient Reels + Automation Grid
- Keep Energy stable with Reactor growth
- Complete easy-aligned quests
- Build to 5 each structure
- Let passive income finish the run

Use this as your default baseline, then adapt by biome and events.
